# Blockquotes

Untuk membuat kutipan blok, tambahkan ">" di depan paragraf.  

#### Contoh :

+ *Sintak menggunakan 3 tanda bintang sebelum dan sesudah teks* 
  
  ```
  TestThe rendered output looks like this:
  > Dorothy followed her through many rooms.
  ```
