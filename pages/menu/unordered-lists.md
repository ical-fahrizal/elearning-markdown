# Unordered Lists

To create an unordered list, add dashes (-), asterisks (*), or plus signs (+) in front of
line items.

#### Contoh :

+ *Syntax 1*
  
  ```
  * First item
  * Second item
  * Third item
  ```
  *hasil :*
  > * First item
  > * Second item
  > * Third item

+ *Syntax 2*
  
  ```
  - First item
  - Second item
  - Third item
  ```
  *hasil :*
  > - First item
  > - Second item
  > - Third item

+ *Syntax 3*
  
  ```
  + First item
  + Second item
  + Third item
  ```
  *hasil :*
  > + First item
  > + Second item
  > + Third item
