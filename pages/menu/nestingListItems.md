# Nesting List Items

Untuk menyarangkan item baris dalam daftar yang diurutkan, identitas item empat spasi atau satu tab. 

#### Contoh :

+ *Syntax 1*
  
  ```
  1. First item
  2. Second item
  3. Third item
     1. Indented item
     2. Indented item
  4. Fourth item
  ```
  *hasil :*
  > 1. First item
  > 2. Second item
  > 3. Third item
  >    1. Indented item
  >    2. Indented item
  > 4. Fourth item
